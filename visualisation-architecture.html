🎨 Extraction de Couleurs

Fonction extractColors() : texte → codes hexadécimaux

flowchart TD
    Input["📥 Input: colors = 'Green and White'"]
    
    Normalize["🔄 ÉTAPE 1: Normalisation
- Minuscules
- Supprimer ponctuation
- Découper en mots
Output: ['green', 'and', 'white']"]
    
    Map["🎨 ÉTAPE 2: Mapping couleurs
colorMap = {
'green': '#16A34A',
'white': '#FFFFFF',
'red': '#DC2626',
... 50+ couleurs }"]
    
    Match["🔍 Recherche correspondances
green → #16A34A
white → #FFFFFF"]
    
    Select["✅ ÉTAPE 3: Sélection
primary = #16A34A (première)
secondary = #FFFFFF (deuxième)"]
    
    Output["📤 Return:
{primary: '#16A34A',
secondary: '#FFFFFF'}"]
    
    Apply["🖌️ Application
setThemeColor('#16A34A')
setSecondaryColor('#FFFFFF')
Génération maillot SVG"]
    
    Input --> Normalize
    Normalize --> Map
    Map --> Match
    Match --> Select
    Select --> Output
    Output --> Apply

    style Output fill:#e8f5e9
