   ↓
2️⃣ Frontend → Backend : Message via CopilotKit
   ↓
3️⃣ GPT-5.2 Analyse :
   - Détecte "FC Barcelone"
   - Extrait infos base connaissances
   - Génère history EN FRANÇAIS
   ↓
4️⃣ GPT-5.2 appelle update_club_info :
   {
     name: "FC Barcelona",

   ↓

   - extractColors("Blue and Red") → {primary: "#1E40AF", secondary: "#DC2626"}

   - setSecondaryColor("#DC2626")
   - Génère maillot SVG
